ВЕРСИЯ V16 — COMIC SHIFT

Что добавлено:
- яркий современный комикс-стиль, крупные подписи и анимации;
- недельный прогноз погоды для Парка «Зарядье» через Open-Meteo;
- офлайн-показ последнего сохранённого прогноза;
- развлекательный ежедневный гороскоп для Романа/Романа II, Василия и Волка;
- отдельный нижний блок математической справедливости;
- автоматическая самопроверка движка расписания.

Проверенная логика перехода:
- сеанс №1: 10:20–10:30;
- сеанс №2: 10:40–10:50;
- сеанс №3: 11:00–11:10;
- ровно с 11:10 уже действуют новые позиции;
- сеанс №4 начинается в 11:20 на новых позициях.

Справедливость:
- первая абсолютная точка равенства: после рабочей смены №1, 13.07.2026 — по 6 сеансов на каждой позиции каждому;
- первое возвращение к равенству: после рабочей смены №5, 21.07.2026 — по 34 сеанса на каждой позиции каждому;
- полный повторяемый календарный цикл: 56 дней / 28 рабочих смен — по 192 сеанса на каждой позиции каждому.

Статистика сохраняется под прежним ключом localStorage ovp-v8-local, поэтому данные предыдущих версий не должны пропасть.
