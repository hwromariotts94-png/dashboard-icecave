ОТМОРОЖЕННЫЕ В ПЕЩЕРЕ — ЛИЧНАЯ ВЕРСИЯ V8

Что работает:
- фиксированный график 2/2 от 13.07.2026;
- ротация каждые 3 сеанса;
- автоматическое определение позиций по московскому времени;
- статистика посетителей и заметки только на одном iPhone;
- график посещаемости;
- погода у Парка Зарядье при наличии интернета;
- офлайн-работа после установки;
- экспорт и импорт резервной копии.

ВАЖНО:
Расписание не хранится в статистике и не зависит от localStorage.
Оно всегда пересчитывается из даты 13.07.2026 и московской даты.
Очистка статистики не изменит смены и ротацию.

Установка через GitHub Pages:
1. Создайте новый публичный репозиторий GitHub.
2. Загрузите в корень все файлы из этой папки.
3. Settings -> Pages -> Deploy from a branch.
4. Branch: main, folder: /root, Save.
5. Откройте выданную ссылку в Safari на iPhone.
6. Подождите 10-20 секунд.
7. Поделиться -> На экран Домой -> Добавить.
8. После этого приложение откроется с иконки и расписание будет работать без интернета.

Данные статистики находятся только на этом iPhone.
Чтобы не потерять их, иногда нажимайте «Экспорт копии» и сохраняйте JSON-файл в iCloud Drive.
Погода без интернета не обновляется, но показывается последнее сохраненное значение.
