ОТМОРОЖЕННЫЕ В ПЕЩЕРЕ — V10

Что изменено:
- Каждый понедельник используется сокращённое расписание: 12:00–16:00, перерыв 16:00–17:00, затем 17:00–21:00.
- В понедельник 24 сеанса по 20 минут.
- В остальные рабочие дни остаётся прежнее расписание на 28 сеансов.
- Каждый рабочий день начинается с новой полной тройки сеансов. Незавершённые тройки между днями не переносятся. Стартовая расстановка сдвигается на одну позицию каждый следующий рабочий день.
- График 2/2 по-прежнему вычисляется от 13.07.2026 по московской дате.
- Старые данные статистики V8 сохраняются, потому что ключ локального хранилища не изменён.

ОБНОВЛЕНИЕ НА GITHUB
1. Скопируйте все файлы V10 в локальную папку dashboard-icecave с заменой.
2. Выполните:
   git add .
   git commit -m "Новая полная тройка каждый день и понедельник 12-21 V10"
   git push origin main
3. На iPhone откройте сайт с интернетом. Если старая версия осталась, закройте приложение, откройте ссылку в Safari и обновите страницу. При необходимости удалите и снова добавьте иконку на экран Домой.


ЛОГИКА V10:
Каждый рабочий день начинается с новой полной тройки сеансов. Остатки троек между днями не переносятся. Стартовая позиция каждого сотрудника сдвигается на одну позицию каждый следующий рабочий день, поэтому нагрузка выравнивается по циклу из 4 рабочих дней. Сокращённый понедельник содержит 24 сеанса и даёт каждому ровно по 6 сеансов на каждой из 4 позиций.
